# Mermoz Sciences Project

## Présentation

Ce projet est un **tableau de bord pédagogique interactif** destiné à accompagner les élèves dans leur apprentissage des mathématiques et des sciences.

Il centralise, dans une seule interface simple et intuitive, les outils numériques essentiels utilisés en classe afin de faciliter l’autonomie, la concentration et l’expérimentation.

L’objectif est de proposer un environnement de travail clair, rapide d’accès et adapté aux besoins des élèves, en particulier dans le cadre de l’enseignement professionnel.

---

## Fonctionnalités principales

### Menu principal interactif

Le tableau de bord propose un accès rapide aux outils suivants :

- Éduc de Normandie
- Son dans la classe
- Timer pédagogique
- Classe Puzzle
- Le Chat IA
- Création de sch́mas (draw.io)
- Programmation Python (Basthon)
- GeoGebra (Classic + 3D)
- Calculatrice graphique NumWorks
- Laboratoire virtuel
- PlayMaths Arcade
- Outil de notation
- Gestion de l’affichage (plein écran / écran fractionné)
- Consultation de la licence Creative Commons

---

## Modes d’affichage

### Mode plein écran

Affichage principal par défaut avec :

- message d’accueil pédagogique
- chargement des outils dans une grande zone centrale
- interface épurée pour la projection en classe

### Mode écran fractionné

Permet de travailler simultanément avec :

- un outil principal à gauche
- le sonomètre de classe en haut à droite
- le timer pédagogique en bas à droite

Ce mode est particulièrement utile pour la gestion de classe.

- ajout de nouveaux outils
- modification des liens
- changement des icônes
- personnalisation du message d’accueil
- adaptation aux besoins d’une autre discipline

## Objectifs pédagogiques

Ce projet vise à :

- favoriser l’autonomie des élèves
- centraliser les ressources utiles
- limiter les pertes de temps
- améliorer la gestion de classe
- encourager l’expérimentation
- valoriser le droit à l’erreur dans les apprentissages
